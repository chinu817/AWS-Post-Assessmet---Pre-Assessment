Q3.Steps to launch EC2 Instance

1.Open the Amazon EC2 console and then choose Launch Instance
2.Choose an Amazon Machine Image(Linux, Windows) and click Select
3.Choose an Instance Type and click Next
4.Configure Instance Details(info like Network, Subnet, File Systems etc)
5.Add Storage
6.Add Tags
7.Give a name to your instance
8.Configure Security Group(info like Protocol, Port Range, Source etc)
9.Choose Review and Launch(review all the configurations you made)
10.Choose Launch
11.Select the check box for the key pair that you created, and then choose Launch Instances
